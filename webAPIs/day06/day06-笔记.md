### 1. 正则表达式  
匹配字符串中字符组合的模式， 正则表达式也是对象  
#### 1.1 什么是正则表达式  
匹配 替换  提取  
#### 1.2 语法  
定义规则  
根据规则查找  
```js
const 变量名 = /表达式/  
regObj.test(被检测的字符串)
regObj.exec(被检测字符串) // 返回数组
```
#### 1.3 元字符  
```js
// 例如匹配小写字母
[a-z]
// 边界符
^
$
// 量词,表示重复次数
{n}
{n,}
{n,m}
*
+
?
// 字符类
[] //匹配字符集合
[a-z]
[0-9]
[A-Z]
[a-z0-9A-Z]
[^a-z] // 集合取反
.
\d
\D // 与\d相反
\w // 匹配单词
\W // 与\w相反
\s // 匹配空格
\S // 与\s相反
```
